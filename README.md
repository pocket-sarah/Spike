## Installation

Install **WHITE SPIKES** with GIT

```bash
git clone https://github.com/madhanmaaz/white-spikes
```
```bash
cd white-spikes
```
- Install Dependencies
```bash
npm install
```

- Run - The default port is `3411` 
```bash
npm start
```
- `http://localhost:3411` Open on your browser
